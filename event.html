// 禁止事項

  • 本館4階、学生寮は立ち入り禁止です。
  • 本校敷地内は全面禁酒・禁煙です。
  • 校舎内は飲食禁止です。飲食物は屋外でお召し上がりください。
  • ごみのポイ捨てはご遠慮ください。

// 屋台について

  • 屋台では金券のみ使用可能です。
  • 金券は本部にて現金でご購入・ご返金いただけます。
  • 屋台でお並びの際は、危ないですので道路にはみ出さないでください。

ご不明な点などございましたら、運営本部またはスーツを着用しているスタッフまでお気軽にお尋ねください。
